## Setup Instructions

### Option 1: GitHub Pages (Recommended)

The website is already deployed using GitHub Pages!

**Repository**: [https://github.com/DhileepKumarN/portfolio](https://github.com/DhileepKumarN/portfolio)

To enable GitHub Pages for this repository:
1. Go to repository settings on GitHub
2. Navigate to "Pages" in the sidebar
3. Under "Source", select the main branch and `/ (root)` folder
4. Click "Save"
5. Your site will be available at `https://dhileepkumarn.github.io/portfolio/`

### Option 2: Local Development

1. Clone this repository:
   ```bash
   git clone https://github.com/DhileepKumarN/portfolio.git
   cd portfolio
   ```

2. Open `index.html` in your web browser, or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

3. Visit `http://localhost:8000` in your browser

## Customization

### Updating Content

- Edit `index.html` to update text content, links, and sections
- Update contact information in the About and Contact sections
- Modify experience entries, skills, and achievements as needed

### Styling

- Edit `styles.css` to change colors, fonts, spacing, and layout
- Main color variables are defined in `:root` at the top of `styles.css`
- Modify `--primary-color`, `--secondary-color`, etc. to change the theme

### Adding Sections

1. Add a new section in `index.html` with a unique `id`
2. Add a corresponding navigation link in the navbar
3. Style the new section in `styles.css`
